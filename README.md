# 🎮 Lista de Exercícios em JavaScript

## 📚 Sobre o projeto

Este projeto foi desenvolvido como parte de uma atividade acadêmica com o objetivo de praticar conceitos básicos de JavaScript, HTML e lógica de programação.

A aplicação consiste em uma página web interativa que reúne diferentes jogos e exercícios, permitindo ao usuário selecionar e executar cada um deles.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* JavaScript

---

## 🎯 Funcionalidades

O sistema contém os seguintes jogos:

### 🎲 Jogo 1: Adivinhação

O computador gera um número aleatório de 1 a 20, e o usuário deve tentar adivinhar. O sistema fornece dicas se o número é maior ou menor.

---

### ✊ Jogo 2: Pedra, Papel ou Tesoura

O usuário escolhe uma opção e o computador gera uma escolha aleatória. O sistema compara os resultados e informa o vencedor.

---

### 🔢 Jogo 3: Tabuada

O usuário informa um número inteiro e o sistema exibe a tabuada de 1 a 10 diretamente na tela.

---

### ⭐ Jogo 4: Triângulo de Asteriscos

O usuário informa a quantidade de linhas e o sistema desenha um triângulo de asteriscos no console e na página.

---

### ➕ Jogo 5: Soma em Série

O programa calcula a soma da série:
1 + 11 + 111 + ... até n termos, conforme valor informado pelo usuário.
